# Contributing Guidelines

Thank you for considering a contribution!

## How to Contribute
1. Open an issue describing your idea or bug report.
2. Fork the repository.
3. Create a feature branch:
    - git checkout -b feature/my-feature
4. Commit clearly:
    - feat: added ODS performance test
    - fix: improved ZIP limit handler
5. Open a pull request.

## Coding Standards
- Shell scripts must run on Bash 4.0+
- PowerShell scripts must support Windows Server 2016+
- Use consistent indentation (spaces only)
- Add comments for complex logic
